Welcome to Simple Onboarding App

To get started, modify this index.html file as you need, calling the component app-root somewhere in the body.

Then simply use the attribute tooltip with attributes step and text in any element to pop up a tooltip for that element.

For example, this tooltip is the result of adding tooltip step="1" to this element.


If you don't provide a step number, all your tooltips will appear all at once, instead of a click-through tour.

This is an example form. This tooltip is the result of adding tooltip step="2" to the div element containing this form.

Have fun!
